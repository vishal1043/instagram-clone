#  Instagram Clone (Flask)

A full-stack Instagram-like web application built using Flask, JWT authentication, and MySQL.

##  Features
- User signup & login (JWT-based)
- Create posts (image upload & URL)
- Like & comment on posts
- Follow / unfollow users
- User search
- Profile page with stats
- Secure backend APIs
- Responsive UI

## Tech Stack
- Backend: Flask, Flask-JWT-Extended
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Auth: JWT
- ORM: Raw SQL (PyMySQL)

##  How to Run
      python -m venv venv
      venv\Scripts\activate
      pip install -r requirements.txt
      python app.py
